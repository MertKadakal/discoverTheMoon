# 🌙 Ay'ı Keşfet (Discover the Moon)

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![Three.js](https://img.shields.io/badge/Graphics-Three.js-black)](https://threejs.org/)
[![Firebase](https://img.shields.io/badge/Database-Firebase-orange)](https://firebase.google.com/)

**Ay'ı Keşfet**, çocuklar ve astronomi meraklıları için tasarlanmış, interaktif ve oyunlaştırılmış bir uzay keşif platformudur. Proje, modern web teknolojilerini kullanarak Ay ve uzay araştırmalarını eğlenceli ve eğitici bir deneyime dönüştürmeyi amaçlamaktadır.

---

## 🚀 Öne Çıkan Özellikler

- **3D Deneyim:** Three.js motoru ile güçlendirilmiş, tarayıcı üzerinden erişilebilen akıcı 3 boyutlu oyun modları.
- **İnteraktif Bilgi Merkezi:** Ay'ın fiziği, jeolojisi ve tarihsel keşifler hakkında görsel odaklı bilgi kartları.
- **Gamified Learning (Oyunlaştırılmış Öğrenme):** Bilginin oyun mekanikleriyle harmanlandığı, ilerlemek için soruların çözülmesi gereken interaktif yapılar.
- **Real-time Scoreboard:** Firebase entegrasyonu ile anlık güncellenen liderlik tablosu.
- **Modern UI/UX:** Karanlık mod temalı, göz yormayan, premium hissiyatlı kullanıcı arayüzü.

---

## 🎮 Oyun Modları

| Oyun | Görev | Mekanik |
| :--- | :--- | :--- |
| **Rover Görevi** | Ay yüzeyindeki numuneleri toplayıp rokete ulaşmak. | 3D Sürüş, Labirent, Bilgi Soruları |
| **Ay İnişi (Landing)** | Meteorlardan kaçarak Moncuk'u kurtarmak. | Yakıt Yönetimi, Refleks, Fizik Simülasyonu |
| **Uzay İstasyonu** | İstasyon içindeki kritik modülleri onarmak. | Bulmaca, Zaman Karşı Yarış, Teknik Görevler |

---

## 🛠️ Teknik Altyapı

- **Frontend:** HTML5, Vanilla CSS3 (Gelişmiş Animasyonlar & Glassmorphism)
- **Engine:** [Three.js r128](https://threejs.org/) (WebGL tabanlı 3D render)
- **Scripting:** Vanilla JavaScript (ES6+)
- **Backend/Database:** Google Firebase (Firestore) - Real-time leaderboard yönetimi için.
- **Design:** AstroDreamer tasarım dili.

## 📦 Kurulum ve Çalıştırma

Projeyi yerel ortamınızda çalıştırmak için şu adımları izleyin:

1. **Depoyu klonlayın:**
   ```bash
   git clone https://github.com/MertKadakal/discoverTheMoon.git
   ```
2. **Firebase Ayarları:**
   - Bir Firebase projesi oluşturun.
   - `firebaseConfig.js` (veya ilgili dosya) içerisine kendi API anahtarlarınızı yerleştirin.
3. **Çalıştırın:**
   - Proje dizinine gidin ve herhangi bir yerel sunucu (örn: VS Code Live Server) ile `index.html` dosyasını başlatın.
